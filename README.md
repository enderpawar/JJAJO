# 🧾 짜조 (JJAJO)

> **말만 하면, AI가 일정을 짜줘요.**  
> 목표를 정하고, 자연어로 계획을 던지면, 캔버스에 바로 반영되는 AI 액션 플래너.

---

## 짜조가 뭐예요?

**짜조**는 "일정 짜줘"에서 나온 이름이에요.  
하고 싶은 걸 **문장으로만 말하면**, Google Gemini AI가 이해해서 **목표·마일스톤·일정**을 만들어 주고, 화면의 **타임라인과 캘린더**에 바로 올려줍니다.

### 핵심 기능

- 📝 **한 줄로 일정 추가** — "내일 오후 3시 회의", "다음 주 월요일부터 금요일까지 운동"처럼 입력하면 자동 파싱·추가
- 🎯 **대화로 목표 설계** — "3개월 안에 토익 900"처럼 말하면 AI가 주차별 계획까지 제안
- 🔄 **역방향 계획(Backwards Plan)** — 마감일에서 거꾸로 밀어넣어서 현실적인 일정 배치
- 🗓️ **멀티데이 일정 지원** — 여러 날에 걸친 일정(프로젝트, 여행 등)을 시작일~종료일로 관리
- 📅 **스마트 일정 배치** — AI가 템플릿 기반으로 휴식 시간까지 자동 배치
- 🔔 **알림 설정** — 일정 시작 전 알림 시간 설정 가능
- ✏️ **드래그 앤 드롭** — 일정 이동·수정을 끌어당겨서 조정
- 🗒️ **메모 기능** — 각 일정에 상세 메모 작성 및 관리

딱딱한 "할 일 목록"이 아니라, **말하는 대로 움직이는 스마트 플래너**에 가깝게 만들었어요.

---

## 어떻게 쓰나요?

### 1️⃣ 로그인

- **Google 계정**으로 로그인합니다. (OAuth 2.0)
- 첫 사용 시 **Gemini API 키**를 한 번 입력해 두면, 이후엔 그 키로 AI 기능이 동작합니다.
- 키는 서버에 저장하지 않는 **BYOK(Bring Your Own Key)** 방식으로 안전합니다.

### 2️⃣ 목표 정하기 (선택)

- **목표** 섹션에서 "이런 걸 하고 싶어요"라고 말하면, AI가 **제목·설명·주당 회차·예상 일수**를 제안합니다.
- 마음에 들면 **「이대로 만들기」**, 아니면 **「다른 방식으로 할게요」**로 대화를 이어가면 됩니다.
- **AI는 제안만, 최종 결정은 사용자**가 하는 구조입니다.

### 3️⃣ 일정 넣기

#### 🪄 매직 바를 통한 자연어 입력

**단일 날짜 일정:**
```
내일 오후 3시 회의
다음 주 월요일 2시간 스터디
```

**멀티데이 일정:**
```
이번 주 월요일부터 금요일까지 운동
다음 달 1일부터 5일까지 부산 여행
```

- AI가 **시작일, 종료일, 시간, 제목**을 자동으로 해석해서 타임라인/캘린더에 바로 추가합니다.
- **템플릿 기반 스마트 배치**: AI가 일정 사이에 적절한 휴식 시간을 자동으로 추가합니다.
- **카테고리 자동 분류**: 입력한 내용을 분석하여 적절한 카테고리(공부, 운동, 업무 등)를 자동 할당합니다.

#### ➕ 직접 추가

- **일정 추가** 버튼으로 상세 정보 입력:
  - 제목, 카테고리
  - 시작일/시간, 종료일/시간 (멀티데이 일정 지원)
  - 알림 설정 (X분 전 알림)
  - 메모 (상세 설명, 링크 등)

### 4️⃣ 캔버스에서 보기·고치기

#### 📊 다양한 뷰

- **세로 타임라인**: 오늘을 중심으로 한 주 단위로 일정이 쭉 나열됩니다.
- **주간 히트맵**: 요일별 일정 밀도를 시각화하여 한눈에 볼 수 있어요.
- **월간 캘린더**: 헤더에서 열면 달력 뷰로 이동.
  - **멀티데이 일정은 날짜를 걸쳐서 표시**됩니다.
  - 날짜를 클릭하면 그날의 상세 일정을 확인할 수 있습니다.
  - 일정 추가, 수정, 삭제 모두 가능합니다.

#### ✨ 인터랙션

- **드래그 앤 드롭**: 일정을 다른 날·다른 시간대로 자유롭게 이동
- **클릭하여 편집**: 일정 카드를 클릭하면 수정 모달이 열립니다.
- **메모 사이드바**: 선택한 일정의 메모를 보고 편집할 수 있습니다.
- **알림 설정**: 각 일정마다 시작 X분 전 알림 설정 가능
- **완료 체크**: 일정을 완료하면 체크 표시로 진행 상황 관리

### 5️⃣ 설정

- **시간대·가용 시간대**를 정해 두면, AI가 그 범위 안에서 일정을 잡아 줍니다.
- **API 키**는 설정에서 변경할 수 있고, 세션/로컬에만 보관됩니다.
- **테마 설정**: 다양한 색상 테마로 캘린더를 꾸밀 수 있습니다.

## 화면 구조 한눈에

```
┌─────────────────────────────────────────────────────────────┐
│  [로고]  짜조        [월간 캘린더] [설정] [로그아웃]           │  ← Header
├─────────────────────────────────────────────────────────────┤
│  [주간 히트맵]  Mon Tue Wed Thu Fri Sat Sun                   │  ← TopTimeline
├─────────────────────────────────────────────────────────────┤
│  [🪄] [예: 내일 오후 3시 회의...        ] [적용]              │  ← Magic Bar
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  09:00  ───  일정 A  ────────────────────                    │
│  10:00       일정 B (3일간) ═══════════════                  │  ← Vertical Timeline
│  14:00            일정 C  ─────────────────────────         │     (멀티데이 지원)
│  ...              [메모] 일정 세부사항                        │  ← MemoSidebar
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  [목표 & 역방향 계획]  목표 카드 · 마일스톤 · Backwards Plan   │  ← Goals Section
└─────────────────────────────────────────────────────────────┘
```

- **헤더**: 로고, 월간 캘린더 열기, 설정, 로그아웃
- **매직 바**: 자연어 입력 → 파싱/추가/수정
- **타임라인**: 일정 시각화와 드래그로 조정 (멀티데이 일정 지원)
- **캘린더**: 월간 뷰에서 멀티데이 일정 시각적 표현
- **메모 사이드바**: 선택한 일정의 상세 메모 관리
- **목표 섹션**: AI와 대화로 만든 목표·역방향 계획

### UI/UX 개선

- **테마 색상 통일**: 전체적인 UI 일관성 향상
- **알림 기능**: `reminderMinutesBefore` 속성으로 일정 전 알림 설정

### 성능 최적화

- **불필요한 useMemo 제거**: `DayDetailPanel` 리렌더링 최적화
- **상태 관리 개선**: `CalendarGrid` 컴포넌트 상태 정리

## 기술 구성 (참고용)

|------|------|
| **프론트엔드** | React 18, TypeScript, Vite, Zustand, Tailwind CSS, Framer Motion |

- **사용자 중심**: AI는 제안만, 최종 결정은 사용자
- **오프라인 우선**: 로컬 상태 관리로 빠른 응답성

## 로컬에서 실행하기

### 1. 백엔드

```bash
cd backend
# GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET 설정 (backend/GOOGLE_OAUTH_SETUP.md 참고)
./mvnw spring-boot:run
```

→ `http://localhost:8080` (Swagger: `/swagger-ui.html`)

### 2. 프론트엔드

```bash
cd frontend
npm install
npm run dev
```

→ 브라우저에서 `http://localhost:5173` 접속 후 로그인 → `/app`에서 플래너 사용

## 주요 기능 미리보기

### 🎯 자연어 일정 추가
```
사용자: "내일 오후 3시부터 5시까지 프로젝트 회의"
AI: ✓ 2026년 2월 22일 15:00-17:00에 "프로젝트 회의" 추가됨
    카테고리: 업무
```

### 📅 멀티데이 일정
```
사용자: "다음 주 월요일부터 금요일까지 운동"
AI: ✓ 2026년 2월 24일 ~ 2월 28일에 "운동" 추가됨
    카테고리: 건강
    매일 같은 시간에 자동 배치
```

### 🔄 역방향 계획
```
목표: "3월 31일까지 토익 900점"
AI 제안:
  ├─ Week 1-2: 기초 문법 정리 (주 3회, 2시간)
  ├─ Week 3-4: 듣기 집중 훈련 (주 4회, 1.5시간)
  ├─ Week 5-6: 모의고사 풀이 (주 5회, 2시간)
  └─ Week 7: 최종 점검 및 복습 (매일, 3시간)
```

### 🗒️ 스마트 메모
```
일정: "프로젝트 회의"
메모:
  - 참석자: 김팀장, 이대리, 박주임
  - 안건: Q1 실적 검토, Q2 계획 수립
  - 준비사항: PPT 자료, 예산안
  - Zoom 링크: https://...
```
