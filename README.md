# 🧾 짜조 (JJAJO)

> **말만 하면, AI가 일정을 짜줘요.**  
> 목표를 정하고, 자연어로 계획을 던지면, 캔버스에 바로 반영되는 액션 플래너.

---

## 짜조가 뭐예요?

**짜조**는 “일정 짜줘”에서 나온 이름이에요.  
하고 싶은 걸 **문장으로만 말하면**, Google Gemini AI가 이해해서 **목표·마일스톤·일정**을 만들어 주고, 화면의 **타임라인과 캘린더**에 바로 올려줍니다.

- 📝 **한 줄로 일정 추가** — “내일 오후 3시 회의”, “다음 주 월요일 2시간 스터디”처럼 입력하면 자동 파싱·추가
- 🎯 **대화로 목표 설계** — “3개월 안에 토익 900”처럼 말하면 AI가 주차별 계획까지 제안
- 📅 **역방향 계획(Backwards Plan)** — 마감일에서 거꾸로 밀어 넣어서 현실적인 일정 배치
- 🗓 **캘린더·타임라인 시각화** — 주간 히트맵, 세로 타임라인, 월간 캘린더로 한눈에 보기
- ✏️ **드래그 앤 드롭** — 일정 이동·수정을 끌어당겨서 조정

딱딱한 “할 일 목록”이 아니라, **말하는 대로 움직이는 플래너**에 가깝게 만들었어요.

---

## 어떻게 쓰나요?

### 1️⃣ 로그인

- **Google 계정**으로 로그인합니다. (OAuth)
- 첫 사용 시 **Gemini API 키**를 한 번 입력해 두면, 이후엔 그 키로 AI 기능이 동작합니다. (키는 서버에 저장하지 않는 BYOK 방식)

### 2️⃣ 목표 정하기 (선택)

- **목표** 섹션에서 “이런 걸 하고 싶어요”라고 말하면, AI가 **제목·설명·주당 회차·예상 일수**를 제안합니다.
- 마음에 들면 **「이대로 만들기」**, 아니면 **「다른 방식으로 할게요」**로 대화를 이어가면 됩니다.  
  → **제안은 AI, 최종 결정은 사용자**가 하는 구조입니다.

### 3️⃣ 일정 넣기

- 상단 **매직 바**에 자연어로 입력합니다.  
  예: `내일 오후 3시 회의`, `다음 주 월요일 2시간 스터디`
- AI가 시간·제목을 해석해서 **타임라인/캘린더에 바로 추가**합니다.
- **수정·삭제**도 매직 바에서 “대화형 수정 모드”로 문장으로 지시할 수 있습니다.

### 4️⃣ 캔버스에서 보기·고치기

- **세로 타임라인**: 오늘을 중심으로 한 주 단위로 일정이 쭉 나열됩니다.
- **주간 히트맵**: 요일별로 얼마나 채워져 있는지 한눈에 볼 수 있어요.
- **월간 캘린더**: 헤더에서 열면 달력 뷰로 이동. 날짜를 눌러 그날 일정을 보고 수정할 수 있습니다.
- **드래그 앤 드롭**으로 일정을 다른 날·다른 시간대로 옮길 수 있습니다.

### 5️⃣ 설정

- **시간대·가용 시간대**를 정해 두면, AI가 그 범위 안에서 일정을 잡아 줍니다.
- **API 키**는 설정에서 변경할 수 있고, 세션/로컬에만 보관됩니다.

```
┌─────────────────────────────────────────────────────────────┐
│  [로고]  짜조        [월간 캘린더] [설정] [로그아웃]           │  ← Header

│  [주간 히트맵]  Mon Tue Wed Thu Fri Sat Sun                   │  ← TopTimeline
├─────────────────────────────────────────────────────────────┤
│  [🪄] [예: 내일 오후 3시 회의...        ] [적용]              │  ← Magic Bar
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  09:00  ───  일정 A  ────────────────────                    │
│  10:00       일정 B  ────────                                │  ← Vertical Timeline
│  14:00            일정 C  ─────────────────────────         │     (오늘 중심 주간)
│  ...                                                         │
│                                                             │
├─────────────────────────────────────────────────────────────┤
```

- **타임라인·캘린더**: 일정 시각화와 드래그로 조정
- **목표 섹션**: AI와 대화로 만든 목표·역방향 계획

## 기술 구성 (참고용)

| 구분 | 내용 |
|------|------|
| **프론트** | React 18, TypeScript, Vite, Zustand, Tailwind, Framer Motion |
| **백엔드** | Spring Boot 3.4, Google OAuth2/OIDC, JPA, H2(로컬) / PostgreSQL(운영) |
| **AI** | Google Gemini (BYOK: 사용자 API 키만 사용, 서버 미저장) |
| **인증** | Google 로그인 → 세션·쿠키, API는 Principal의 userId로 식별 |

상세 아키텍처·보안(OIDC)·비동기 로깅·테스트 전략은 아래 **참고 문서**의 기존 README와 가이드를 보면 됩니다.

## 로컬에서 실행하기

### 1. 백엔드

```bash
cd backend
# GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET 설정 (backend/GOOGLE_OAUTH_SETUP.md 참고)
./mvnw spring-boot:run
```

→ `http://localhost:8080` (Swagger: `/swagger-ui.html`)

### 2. 프론트엔드

```bash
cd frontend
npm install
npm run dev
```

→ 브라우저에서 `http://localhost:5173` 접속 후 로그인 → `/app`에서 플래너 사용
